Mon Portfolio

  • Portfolio personnel

  • Année : 2024

  • Voir le code

  • Rôle : Développeuse Front

  • Responsabilité : Conception graphique / Développement / SEO

J'ai opté pour une conception de ce portfolio centrée sur l'éco-conception et l'accessibilité. J'ai veillé à utiliser une sémantique HTML appropriée pour offrir une expérience utilisateur optimale, tout en respectant les principes d'accessibilité et les meilleures pratiques en matière d'éco-conception.

Pour assurer la qualité de ce projet, j'ai consulté des référentiels comme le RGAA et le guide de bonnes pratiques RGESN, et utilisé des outils d'évaluation tels que W3C, Stylus, HeadingsMap, Web developer et Wave Evaluation Tool.
